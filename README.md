# React-demo-myMemo
ps: 原示例地址：https://github.com/damonare/memos

一个使用react + react-router + redux的示例

## 主要使用的技术：
* create-react-app脚手架
* react
* react-router@4.2.0
* react-router-dom@4.2.2
* redux@3.7.2
* react-redux@5.0.6
* prop-types@15.6.0
* less@2.7.3
* less-loader@4.0.5



## memo有哪些功能？

* 添加事项
* 修改事项
* 删除事项
* 搜索事项（模糊搜索待更新）
* 本地存储事项
